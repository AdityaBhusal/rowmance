# Rowmance

A straightforward, lightweight theatre ticketing and seat management system.

**Rowmance** is a web-based tool designed to handle the basics of a box office: selecting seats, managing availability, and processing bookings without the usual overhead of enterprise software.

---

## 🛠 Features

* **Interactive Seating Map:** A visual layout of the house for easy seat selection.
* **Live Availability:** Real-time updates to prevent double-booking.
* **Booking Management:** Simple CRUD operations for tickets and customer info.
* **Tiered Pricing:** Support for different seat categories (VIP, Standard, etc.).
* **Admin Dashboard:** Basic reporting on sales and remaining inventory.

## 💻 Tech Stack

* **Frontend:** [Angular]
* **Backend:** [.NET (10) & FastAPI]
* **Database:** [PostGreSQL]
